Mrecp - MRE console primitive

Basic app - console for MRE platform mobile phone (including Nokia S30+). Tested on Nokia 225 with display resolution 240x320. For using with Nokia mobile phone, app must be signed with IMSI (your SIM card) code. https://vxpatch.luxferre.top. Application file - "Mrecp.vxp".
